# Student-Data-Entry-System.
ESP32-Based Student Database Management System using C++ and CSV file storage for searching, updating, and managing student records.
## Database File

The project uses a CSV file (`cp project data.csv`) as a lightweight database.

Fields:
- RollNo
- Name
- FatherName
- Class
- Section
- University

The application performs:
- Record insertion
- Record searching
- Record updating
- Duplicate Roll Number detection
